Overview

This API acts as a clean, standardized gateway to the TripAdvisor Content API. Unlike scraping-based alternatives, every response comes directly from TripAdvisor's official infrastructure — meaning no breakage, no blocks, no stale data.

ℹ️
This is a BYOK (Bring Your Own Key) API. You need a free TripAdvisor Partner API key to use it. See the Quick Start section to get yours in under 5 minutes.
Base URL https://tripadvisor-gateway-api1.p.rapidapi.com

All requests use HTTPS and return JSON. The TripAdvisor key is passed in every request via the X-TripAdvisor-Key query parameter.

02

Authentication

🔐 RapidAPI Key

Authenticate all requests with your RapidAPI subscription key in the header.

HEADER
X-RapidAPI-Key
VALUE
YOUR_RAPIDAPI_KEY

🗝️ TripAdvisor Partner Key (BYOK)

Pass your personal TripAdvisor Content API key as a query parameter in every request. It's free to obtain — see Quick Start below.

QUERY PARAM
X-TripAdvisor-Key
VALUE
YOUR_TA_PARTNER_KEY
03

Quick Start

1

Create a TripAdvisor Developer Account

Go to tripadvisor.com/developers and sign up for free. No credit card required.

2

Generate your API Key

In the developer dashboard, navigate to Credentials and create a new key.

3

Subscribe on RapidAPI

Choose a plan that fits your usage. The Free plan includes 500 requests/month — enough to test everything.

4

Make your first request

Pass both keys and you're live. See the example below.

curl "https://tripadvisor-gateway-api1.p.rapidapi.com/search?searchQuery=Copacabana&category=restaurants&X-TripAdvisor-Key=YOUR_TA_KEY" \ -H "X-RapidAPI-Key: YOUR_RAPIDAPI_KEY"
import requests headers = { "X-RapidAPI-Key": "YOUR_RAPIDAPI_KEY" } params = { "searchQuery": "Copacabana", "category": "restaurants", "X-TripAdvisor-Key": "YOUR_TA_KEY" } response = requests.get("https://tripadvisor-gateway-api1.p.rapidapi.com/search", headers=headers, params=params) print(response.json())
const res = await fetch( "https://tripadvisor-gateway-api1.p.rapidapi.com/search?searchQuery=Copacabana&category=restaurants&X-TripAdvisor-Key=YOUR_TA_KEY", { headers: { "X-RapidAPI-Key": "YOUR_RAPIDAPI_KEY" } } ); const data = await res.json(); console.log(data);

Error Handling

400
BAD_REQUEST — Missing required parameter
401
INVALID_TA_KEY — TripAdvisor key missing, invalid, or expired
401
INVALID_RAPIDAPI_KEY — RapidAPI subscription key is invalid
429
TA_RATE_LIMIT — TripAdvisor key exceeded rate limit. Retry after 1 second.
429
PLAN_LIMIT — RapidAPI plan monthly request limit reached
403
PLAN_REQUIRED — /location/intelligence requires Ultra or Mega plan
500
UPSTREAM_ERROR — TripAdvisor API returned an unexpected error. Retry with exponential backoff.
